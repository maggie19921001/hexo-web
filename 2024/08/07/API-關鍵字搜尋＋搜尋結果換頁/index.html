使用TDX API關鍵字搜尋，並依照分類執行換頁顯示結果

  1. 監聽搜尋按鈕並關閉預設提交機制
  2. 先判斷下拉選單的分類為何，再抓取資料
  3. 網址列中filter代入使用者輸入文字(須先觀察網址列規則)
  4. success中將搜尋結果存儲在localStorage，並跳轉至結果頁面

搜尋頁

  1. 從localStorage中獲取搜尋結果
  2. 設定組成畫面內容functiondisplayResults(分三個分類的內容，個別帶入圖片及內容)
  3. 加入監聽結果
